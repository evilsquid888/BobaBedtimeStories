# CLAUDE.md — Project Guide for BobaBedtimeStories

## What This Project Is
A collection of 20 illustrated bedtime stories for ages 9-12, set in Taiwan's vibrant food culture. Every main character is a sentient Taiwanese dessert ingredient — tapioca pearls, taro, mochi, shaved ice, and more. Stories blend humor, heart, and light Mandarin vocabulary into cozy nighttime adventures.

## Writing Rules (Quick Reference)
- Each story: ~1800-2000 words
- Fun must start within the first 150 words
- A laugh line in every major scene
- Warm narrator voice — playful, silly, cozy
- Characters have vivid physical descriptions tied to their ingredient
- "Getting eaten" endings are JOYFUL and FUNNY, never scary
- Each story ends with a Goodnight Blessing (soft, sleepy, warm)
- 3 AI image prompts per story (key scene, character portrait, closing scene)
- Mandarin terms on first use: **bold Chinese** (pinyin) = English

## Mandarin Integration
- Use Mandarin naturally — sprinkle terms in dialogue and narration
- Always provide pinyin and English on first use
- Key glossary terms are in STYLE_GUIDE.md

## Cross-Over Characters
Some stories feature guest characters from sibling universes:
- **Penguinsnowboard** penguins: Piper Paddlefoot, Barnaby Beaksworth, Willa Wobble, Finnegan Flipsworth
- **Uncle Piggle Wiggle** characters: Uncle Piggle Wiggle, Timmy

## Tone
Think: Roald Dahl meets a Taiwanese night market. Silly but never mean. Warm but never saccharine. Educational but never preachy.
